# TBB Widgets

Public hosting for The Bigger Box chat widgets.
Served via GitHub Pages at https://superbazz.github.io/tbb-widgets/

## Widgets

| File | URL | Description |
|---|---|---|
| moria-mistral.html | https://superbazz.github.io/tbb-widgets/moria-mistral.html | Moria client chat widget (Mistral) |
| tbb-coachy-embed.html | https://superbazz.github.io/tbb-widgets/tbb-coachy-embed.html | TBB public widget — Coachy embed via Make webhook |

## Embedding in Coachy

In any Coachy lesson, go to the **iFrame** tab and paste:

```
https://superbazz.github.io/tbb-widgets/tbb-coachy-embed.html
```

Set height to at least 600px for a comfortable chat experience.
